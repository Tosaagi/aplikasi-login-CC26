# aplikasi-login-CC26
Sebuah aplikasi dengan fitur login (dibuat sebagai bentuk latihan Codelab Pengarsipan versi Perangkat Lunak pada Coding Camp 2026)

Alur Fitur Login:
1. Memasukkan email dan password
2. Memvalidasi masukan.
3. Memvalidasi pengguna dengan deteksi wajah.
4. Memasukkan nama ibu untuk validasi.
5. Menampilkan halaman home.
